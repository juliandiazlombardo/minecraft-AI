# MY_minecraft_ai

Agentes de inteligencia artificial para Minecraft (basados en Python y LLMs).
Este proyecto separa habilidades atómicas (`tools` en Python) de estrategias complejas (`skills` en Markdown) para manejar de forma robusta la integración de IAs en Minecraft.

## Requisitos de inicio
- Python 3.10+
- Node.js instalado en el sistema
- Entorno virtual (recomendado)

## Instalación
```bash
pip install -r requirements.txt
```
