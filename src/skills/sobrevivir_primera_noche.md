# Skill: Sobrevivir la primera noche

## Objetivo
El objetivo es encontrar refugio, conseguir cama y evitar monstruos cuando se pone el sol.

## Consideraciones
1. Lo primero es hacer herramientas de madera.
2. Si ves una oveja, usa tool `atacar` para conseguir lana.
3. Construye un refugio de tierra antes de anochecer.
